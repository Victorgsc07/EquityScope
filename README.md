# EquityScope

EquityScope es una plataforma web enfocada en la consulta, análisis y visualización de información bursátil. El sistema permite revisar datos históricos de acciones, consultar información relevante por ticker y generar predicciones básicas utilizando modelos como promedio móvil y regresión lineal.

El proyecto está pensado como una herramienta clara, minimalista y profesional para analizar el comportamiento de activos financieros de forma visual e intuitiva.

---

## Descripción general

EquityScope permite al usuario seleccionar un ticker bursátil y visualizar información relacionada con su comportamiento histórico. Además, incluye una sección de predicciones donde se aplican métodos estadísticos sencillos para estimar tendencias futuras.

El sistema está dividido en dos partes principales:

- **Frontend:** Interfaz visual donde el usuario consulta datos, selecciona tickers y observa gráficas.
- **Backend:** Servidor encargado de procesar solicitudes, consultar la base de datos y entregar la información al frontend.

---

## Funcionalidades principales

- Consulta de datos bursátiles por ticker.
- Visualización de precios históricos.
- Gráficas interactivas para analizar tendencias.
- Predicción mediante promedio móvil.
- Predicción mediante regresión lineal.
- Menú desplegable para seleccionar acciones disponibles.
- Diseño minimalista y profesional.
